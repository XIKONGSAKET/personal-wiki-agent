# Sun Yimin Perspective Skill

孙一民思维顾问 Skill - 以华南理工大学孙一民教授的思维方式提供城市设计、建筑营建、制度设计咨询。

## 文件结构

```
.claude/skills/sun-yimin-perspective/
├── CLAUDE.md                 # Skill 入口文档
├── README.md                 # 本文件
├── persona-prompt.md         # Persona Prompt（核心配置文件）
└── references/
    ├── cognitive-profile.md  # 认知画像
    └── research/             # 调研原始输出
        ├── 01-writings.md
        ├── 02-conversations.md
        ├── 03-expression-dna.md
        ├── 04-external-views.md
        ├── 05-decisions.md

```

## 使用方法

### 方式一：直接加载 Skill
```bash
/claude skill add sun-yimin-perspective
```

### 方式二：在对话中引用
在需要孙一民视角时，直接描述问题即可。

### 方式三：特定指令
- `/孙一民视角 [问题]` - 标准咨询
- `/孙一民-诊断 [项目]` - 问题诊断
- `/孙一民-案例 [类型]` - 案例查询
- `/孙一民-制度 [设计]` - 制度建议
- `/孙一民-批判 [方案]` - 批判反思

## 核心能力

- ✅ 城市设计项目咨询（概念策划、实施路径、技术统筹）
- ✅ 建筑营建咨询（全生命周期、运营前置、精明投资）
- ✅ 制度设计（总设计师制、多部门协同）
- ✅ 学术讨论（紧凑城市、精明增长、人民城市）
- ✅ 方案批判（诊断问题、识别风险）

## 数据来源

本 Skill 基于以下资料蒸馏：
- 《精明营建：城市大型复杂工程理论与实践》(2025)
- 《总设计师制与城市设计实施》(2021)
- 孔宇航对话孙一民访谈(2024)
- 学生论文（应亦宁、叶伟康等）
- 广州大学城、琶洲西区等项目案例
